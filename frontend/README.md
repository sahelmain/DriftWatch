# DriftWatch — Frontend

React 19 + TypeScript + Vite dashboard for [DriftWatch](../README.md).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Env

See [`.env.production`](.env.production) and the root [README](../README.md#configuration). Set `VITE_API_URL` to your API base (e.g. `http://localhost:8000/api` or `/api` on Vercel with rewrites).
